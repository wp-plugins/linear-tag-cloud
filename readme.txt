== Description ==

A simple and clean tag cloud with a list of tags in percentual lines. Suitable for any theme because have measures in percentage and custom options for max tag number; background, border and text color of the bars.

== Installation ==

1. Download the .zip file from the official repository and unzip it
2. Upload the folder `linear-tag-cloud` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the widget in the sidebar areas and configure it whit max tag number, border color, bars color, link color
5. Done!

== Frequently asked questions ==

= Is it possibile to configure it? =
Yes, you can configure max number of tags; background, text and border color of the bars

= Can I change the style? =
You can apply whatever CSS rules to the elements of the widget. The main class of the `div` contain the tag cloud is `.linear-tag-cloud`.
